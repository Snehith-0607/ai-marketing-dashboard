Build a modern AI-powered Business Intelligence web application called:

"Conversational AI for Instant Business Intelligence Dashboards"

The goal of this application is to allow non-technical users to generate interactive data dashboards using natural language prompts.

The application should feel like a combination of ChatGPT + a Business Intelligence tool (like Tableau or PowerBI).

Use modern UI design with a clean SaaS aesthetic.

Tech stack requirements:
Frontend: React or Next.js
Styling: TailwindCSS
Charts: Recharts or Chart.js
The UI must be responsive and professional.

---

MAIN APPLICATION ARCHITECTURE

The application must contain four main pages:

1. Landing Page
2. Chat Interface (main AI interaction area)
3. Dashboard Page (full analytics view)
4. CSV Upload Page

---

1. LANDING PAGE

Create a modern SaaS landing page for the AI dashboard product.

Sections required:

Hero Section
Headline:
"Conversational AI for Instant Business Intelligence Dashboards"

Subheadline:
Generate powerful analytics dashboards instantly using natural language. No SQL. No complex BI tools.

Buttons:
Try Demo
View GitHub

Hero Visual:
Show a preview image of a dashboard with charts.

---

Problem Section

Explain the business problem:

Most companies rely on technical teams to generate reports and dashboards. Business users cannot easily access insights because they do not know SQL or how to configure BI tools.

---

Solution Section

Explain the product solution:

Users can simply ask questions in natural language, and the AI system will generate dashboards automatically.

Example queries:

"Show monthly sales revenue by region"

"Compare product category sales"

"Display revenue growth over time"

---

Features Section

Display feature cards with icons.

Features include:

Natural Language Analytics
Ask questions about your data like a conversation.

AI Chart Selection
The system automatically chooses the best visualization.

Interactive Dashboards
Charts support hover tooltips and filtering.

Follow-up Queries
Users can refine dashboards with additional prompts.

CSV Upload Support
Users can upload their own datasets.

Real-Time Insights
Dashboards are generated instantly.

---

Tech Stack Section

Display icons or labels for:

Next.js
Gemini API
FastAPI or Node backend
Recharts
CSV / SQL database

---

Call to Action

Button:
Start Exploring Your Data

3. DASHBOARD PREVIEW (INSIDE CHAT)

When the AI generates a response, display a dashboard preview card inside the chat.

This preview must include:

Maximum 3 charts
Bar chart
Line chart
Pie chart

Below the charts show AI-generated insight text.

Example:

Insight:
Revenue increased by 24% in Q3.
The East region is the top-performing region.

At the bottom include a button:

"Open Full Dashboard"
